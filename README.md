## 기능
- PARSE 버튼을 누르면 **시간**과 함께 **급식메뉴**가 표시됨.
- 알레르기표시
- 당장 바로 다음 급식만 표시.

|   |끝 시간|index|
| --- | --- |--- |
|조식 끝|7:30 |1|
|중식 끝|13:10|2|
|석식 끝|18:10|0|
